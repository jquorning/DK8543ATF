
# WeDoNu - We Do Nu
Collaborative activity management tool

## Status
Compileing - Exception in web request

## Dependencys
* AWS - Ada Web Server
* SQLite3 - Simple Components by D. Kazakow
* readline - GNATColl.Readline

## Configuring
```sh
$ ed var/PROGRAM_VERSION
$ ed var/PROGRAM_NAME
```

## Building
```sh
$ make
```

or

```sh
$ gprbuild wedonu.gpr
```

## Running
```sh
$ cd var/
$ ../binary/wedonu
```
